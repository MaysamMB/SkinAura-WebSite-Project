# SkinAura - Premium Skincare & Beauty Website

![SkinAura Banner](proj_img/site_icon.jpg)

## Overview
SkinAura is a fully responsive e-commerce website for premium skincare and beauty products. It features a modern design with both frontend and backend functionality, including user authentication, product management, and shopping cart system.

## Features

### Frontend Features
- **Responsive Design** - Works on all devices (mobile, tablet, desktop)
- **Modern UI/UX** - Clean, elegant interface with hover effects
- **Multi-section Layout**:
  - Hero section with call-to-action buttons
  - Featured products gallery
  - Services/pricing sections
  - Team members showcase
  - Recent news/blog
  - Contact/footer section
- **Interactive Elements**:
  - Dropdown navigation menu
  - Social media integration
  - Image hover effects
  - Smooth scrolling navigation

### Backend Features
- **User Authentication System** (login_form.php)
- **Full CRUD Operations**:
  - Create/Add products (insert_form.php)
  - Read/View products (select_form.php)
  - Update products (update_form.php)
  - Delete products (delete_form.php)
- **Database Management** (conn.php)
- **Shopping Cart** (cart_page.php)
- **Error Handling** (error_page.php)
- **Modular Navigation** (menu.php)

## Installation & Setup

### Prerequisites
- [XAMPP](https://www.apachefriends.org/download.html) (Apache + MySQL + PHP)
- Web browser (Chrome, Firefox, etc.)
- Code editor (VS Code, Sublime, etc.)

### Step-by-Step Installation

#### Step 1: Install XAMPP
1. Download XAMPP from [Apache Friends](https://www.apachefriends.org/)
2. Run the installer and select:
   - Apache
   - MySQL
   - PHP
   - phpMyAdmin
3. Install to `C:\xampp\` (default location)

#### Step 2: Start XAMPP Services
1. Open XAMPP Control Panel
2. Start **Apache** server
3. Start **MySQL** database
4. Verify services are running (green indicators)
